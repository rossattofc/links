            <h2>O Que é Criatividade na Prática?</h2>
            <div class="main-text-content">
                <p>Frequentemente, a criatividade é vista como um talento inato. No entanto, ela é, acima de tudo, uma habilidade que pode ser sistematicamente desenvolvida. Em essência, <strong>ser criativo é a capacidade de fazer novas conexões a partir de elementos que já existem</strong>.</p>
                <p>No dia a dia, a criatividade se manifesta de formas simples e práticas: um colaborador que otimiza uma planilha, uma equipe que encontra uma forma mais eficiente de atender um associado ou um gestor que propõe um novo fluxo para um processo interno. Todos esses são atos criativos que geram valor.</p>
                 <p>É útil diferenciar a "criatividade de impacto" (grandes invenções) da "criatividade cotidiana". Nosso foco é nesta última: a habilidade de resolver pequenos e médios problemas de forma original e eficiente.</p>
                <div class="highlight-box">
                    A criatividade não exige a invenção de algo totalmente novo. Ela floresce na recombinação de ideias, processos e ferramentas existentes para gerar um resultado útil e original.
                </div>

            <h2>Barreiras Comuns à Criatividade</h2>
            <div class="main-text-content">
                <p>Identificar os obstáculos que inibem o pensamento criativo é o primeiro passo para superá-los. Muitas dessas barreiras são hábitos ou pressões do ambiente de trabalho que podem ser conscientemente gerenciados.</p>
                
                <div class="barrier-list">
                    <div class="barrier-item">
                        <h4>Medo do Julgamento e da Falha</h4>
                        <p>O receio de que uma ideia seja criticada ou de cometer um erro é um dos maiores inibidores. A criatividade exige experimentação, e a experimentação implica em possíveis falhas.</p>
                    </div>
                    <div class="barrier-item">
                        <h4>Excesso de Rotina e Pensamento Rígido</h4>
                        <p>Fazer as coisas "do mesmo jeito de sempre" é eficiente, mas limita a exploração de novas abordagens. A rigidez mental nos impede de ver conexões alternativas.</p>
                    </div>
                    <div class="barrier-item">
                        <h4>Falta de Tempo e Pressão por Resultados Imediatos</h4>
                        <p>A criatividade precisa de um período de incubação. Agendas sobrecarregadas e a pressão por produtividade constante podem sufocar o tempo necessário para refletir e conectar ideias.</p>
                    </div>
                     <div class="barrier-item">
                        <h4>Falta de Estímulo e Ambiente Inadequado</h4>
                        <p>Um ambiente que não valoriza novas ideias, que não oferece diversidade de pensamento ou que não provê segurança psicológica desestimula a participação e a colaboração criativa.</p>
                    </div>

            function showScreen(screenId) {
                const footer = document.getElementById('main-footer');
                
                document.querySelectorAll('.screen').forEach(screen => {
                    screen.classList.remove('active');
                });
                
                const activeScreen = document.getElementById(screenId);
                if (activeScreen) {
                    activeScreen.classList.add('active');
                }

                if (screenId === 'welcome-screen') {
                    footer.style.display = 'flex';
                    if (document.getElementById('quiz-results')) {
                        document.getElementById('quiz-results').style.display = 'none';
                    }
                    if(document.getElementById('finish-module-btn')) {
                        document.getElementById('finish-module-btn').style.display = 'none';
                    }
                    if(document.getElementById('quiz-nav-buttons')) {
                        document.getElementById('quiz-nav-buttons').style.display = 'flex';
                    }
                    document.querySelectorAll('.options label').forEach(label => {
                        label.classList.remove('selected', 'correct', 'incorrect');
                    });
                    document.querySelectorAll('input[type="radio"]').forEach(radio => {
                        radio.checked = false;
                    });
                } else {
                    footer.style.display = 'none';
                }
                
                window.scrollTo(0, 0);
            }
            
            function buildTimeline(containerId, data) {
                const container = document.getElementById(containerId);
                if (!container) return;
                container.innerHTML = '';
                
                data.forEach((item, index) => {
                    const positionClass = index % 2 === 0 ? 'left' : 'right';
                    const timelineItem = document.createElement('div');
                    timelineItem.className = `timeline-item ${positionClass}`;
                    
                    timelineItem.innerHTML = `
                        <div class="timeline-header">
                            <h3><span class="timeline-icon">${item.icon}</span>${item.title}</h3>
                        </div>
                        <div class="timeline-content">
                            <div class="timeline-content-inner">
                                ${item.content}
                            </div>
                        </div>
                    `;
                    container.appendChild(timelineItem);
                });
            }

            function toggleTimelineItem(header) {
                const timelineItem = header.closest('.timeline-item');
                const content = timelineItem.querySelector('.timeline-content');
                const isAlreadyActive = timelineItem.classList.contains('active');

                // Fecha todos os outros itens abertos
                document.querySelectorAll('.timeline-item.active').forEach(activeItem => {
                    if (activeItem !== timelineItem) {
                        activeItem.classList.remove('active');
                        activeItem.querySelector('.timeline-content').style.maxHeight = null;
                    }
                });

                if (!isAlreadyActive) {
                    timelineItem.classList.add('active');
                    content.style.maxHeight = content.scrollHeight + "px";
                } else {
                    timelineItem.classList.remove('active');
                    content.style.maxHeight = null;
                }
            }

            function showQuizResults() {
                let correctCount = 0;
                const questions = document.querySelectorAll('.question-container');
                
                questions.forEach(question => {
                    const optionsDiv = question.querySelector('.options');
                    const correctAnswer = optionsDiv.getAttribute('data-answer');
                    const selectedRadio = question.querySelector('input[type="radio"]:checked');
                    
                    optionsDiv.querySelectorAll('label').forEach(l => {
                        l.classList.remove('correct', 'incorrect');
                    });

                    if (selectedRadio) {
                        const selectedLabel = selectedRadio.parentElement;
                        if (selectedRadio.value === correctAnswer) {
                            correctCount++;
                            selectedLabel.classList.add('correct');
                        } else {
                            selectedLabel.classList.add('incorrect');
                            optionsDiv.querySelector(`input[value="${correctAnswer}"]`).parentElement.classList.add('correct');
                        }
                    }
                });

                const totalQuestions = questions.length;
                const percentage = (correctCount / totalQuestions) * 100;
                const resultsEl = document.getElementById('quiz-results');
                resultsEl.textContent = `Você acertou ${correctCount} de ${totalQuestions} perguntas (${percentage.toFixed(0)}%).`;
                resultsEl.style.display = 'block';
                
                document.getElementById('quiz-nav-buttons').style.display = 'none';
                document.getElementById('finish-module-btn').style.display = 'inline-block';

                // ENVIA A NOTA PARA O SCORM
                if(typeof scorm !== 'undefined') {
                    scorm.complete(percentage.toFixed(0));
                }
            }

            // --- INICIALIZAÇÃO E EVENT LISTENERS ---

                    const header = e.target.closest('.timeline-header');
                    if (header) {
